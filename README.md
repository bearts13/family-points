# 家庭积分成长系统

🏦 每天进步一点点，积分攒起来！

一个帮助家长管理孩子日常行为的积分系统，支持赚取积分、消费积分、利息结算等功能。

## 功能特点

- 🧒 **儿童模式**：完成任务赚取积分
- 👨‍👩‍👧 **家长模式**：管理规则、查看报表
- 📱 **移动端适配**：支持手机和平板访问
- ☁️ **云端同步**：数据实时保存到云端
- 🔗 **家庭绑定**：通过邀请码加入家庭

## 使用说明

1. 家长注册账户，系统会自动生成家庭邀请码
2. 孩子通过邀请码加入家庭
3. 完成任务赚取积分，积分可兑换娱乐时间

## 技术栈

- HTML + CSS + JavaScript
- Supabase（云端数据库）
